Summary: Latitude vs. X

The purpose of this project is to test our HTML and CSS skills by designing a webpage. This webpage will showcase the data of our project

The data and visualizations on this website compare various weather attributes to how far cities are from the equator. We used a weather API to assemble a dataset of over 500 cities. The visualizations were made using Matplotlib.
